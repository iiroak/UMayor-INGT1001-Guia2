Ejercicio 1: El Validador de Tipos Inquebrantable

Pide al usuario su edad. Si escribe letras (ej: "veinte"), el programa debe atrapar el error, no caerse, y volver a pedir el dato hasta que ingrese un número entero válido.

Problema: int() colapsa si le pasas letras. Esto es una vulnerabilidad básica.
Solución: Un ciclo infinito controlado que solo se rompe si la conversión es exitosa.

  • Inicia un ciclo while True:.
  • Intenta (try) pedir el dato y convertirlo a int.
  • Si funciona, usa break para salir del ciclo.
  • Si falla (except ValueError), muestra un mensaje de error amigable.
  • try: Código que podría fallar.
  • except ValueError: Qué hacer si falla específicamente por un error de valor/tipo.
def main():
    # Infinite loop to enforce valid input
    while True:
        try:
            # Attempt to get input and cast to integer
            age = int(input("Ingrese su edad: "))
            break # Exit loop if cast is successful
        except ValueError:
            # Catch the error to prevent application crash
            print("Error TIC: Entrada inválida. Por favor ingrese un número entero.")
            
    print(f"Registro exitoso. Edad confirmada: {age}")

# Execute the main function
main()
Ejercicio 2: Analizador de Textos (Casting Dinámico)

Pide una frase al usuario. Valida que no esté vacía ni que sean solo espacios. Luego, cuenta cuántas vocales tiene iterando sobre ella.

Los strings son colecciones. Debemos limpiar la entrada de espacios accidentales para no falsear datos y asegurar que el usuario escribió algo.

  • Usa un While para obligar el ingreso. El método .strip() limpia espacios. Si el largo es 0, vuelve a pedir.
  • Usa un FOR para recorrer letra por letra.
  • Usa el operador in para verificar si la letra es una vocal.
  • string.strip(): Limpia bordes.
  • letra.lower() in "aeiou": Chequeo múltiple ultra rápido.
def main():
    # Require non-empty input
    while True:
        text = input("Ingrese una frase para analizar: ").strip()
        if len(text) > 0:
            break
        print("Error: El texto no puede estar vacío.")

    vowel_count = 0
    # Iterate through each character in the string
    for char in text:
        # Check against a subset of characters safely
        if char.lower() in "aeiouáéíóú":
            vowel_count += 1
            
    print(f"Análisis completo: Se detectaron {vowel_count} vocales.")

main()
Ejercicio 3: Generador Seguro de Descuentos

Pide el monto de una compra (Float). Valida que sea estrictamente mayor a 0. Si es > 100000, aplica 15% descuento; si es > 50000, 5%; si no, 0%. Muestra el total final.

Mezclamos try/except (para evitar que escriban letras) con validación lógica (para evitar montos negativos o cero que no tienen sentido comercial).

  • El bloque Try maneja la arquitectura de software. Un IF interno maneja la lógica de negocio (valor > 0).
  • Luego, aplicamos ELIFs ordenados de mayor a menor para los tramos.
  • Uso combinado de jerarquía de if para evitar fallos lógicos.
def main():
    amount = 0.0
    # Input validation phase
    while True:
        try:
            amount = float(input("Ingrese el monto de la compra: $"))
            if amount > 0:
                break
            else:
                print("Lógica comercial: El monto debe ser mayor a cero.")
        except ValueError:
            print("Error TIC: Ingrese un formato numérico válido.")

    # Business logic phase
    discount = 0.0
    if amount > 100000:
        discount = amount * 0.15
    elif amount > 50000:
        discount = amount * 0.05

    final_total = amount - discount
    print(f"Total a pagar: ${final_total} (Descuento aplicado: ${discount})")

main()
Ejercicio 4: El Sistema de Alarmas (Filtro con Continue)

Simula el procesamiento de 10 sectores (1 al 10). Si el sector es múltiplo de 3, ignóralo (salta la iteración). Imprime los sectores procesados.

Optimización de recursos. Si un dato no nos sirve, no lo procesamos ni lo imprimimos, pasamos de inmediato al siguiente usando el operador Módulo para detectar múltiplos.

  • Ciclo For con Range(1, 11).
  • Condición: if sector % 3 == 0: continue.
  • % (Módulo) combinado con continue.
def main():
    print("Iniciando escaneo de sectores...")
    
    # Iterate through sectors 1 to 10
    for sector in range(1, 11):
        # Skip processing if sector is a multiple of 3
        if sector % 3 == 0:
            print(f"Sector {sector} ignorado por mantención.")
            continue
            
        print(f"Procesando datos seguros del sector {sector}...")

main()
Ejercicio 5: La Factorial (Acumulador Multiplicativo)

Pide un entero positivo seguro. Calcula su factorial (ej: 4! = 1 * 2 * 3 * 4). Usa un FOR para la matemática y manejo de errores para el input.

Un acumulador multiplicativo NO puede iniciar en 0, de lo contrario todo el resultado se anulará (x * 0 = 0). Debe iniciar en 1.

  • Valida el input para enteros positivos.
  • Inicia resultado = 1.
  • Ciclo for iterando desde 1 hasta N inclusive (range(1, n+1)).
  • range(1, n + 1) para asegurar que el último número se multiplique.
def main():
    # Enforce positive integer input
    while True:
        try:
            num = int(input("Ingrese un número entero positivo para el cálculo: "))
            if num > 0:
                break
            print("El número debe ser mayor a cero.")
        except ValueError:
            print("Entrada inválida. Se requiere un entero.")

    # Multiplicative accumulator starts at 1
    factorial = 1
    
    # Calculate factorial
    for i in range(1, num + 1):
        factorial *= i
        
    print(f"El factorial de {num} es: {factorial}")

main()

Ejercicio 6: Sistema ATM con Bloqueo de Seguridad

PIN correcto: "9999". El usuario tiene 3 intentos. Si ingresa texto, cuenta como intento fallido sin colapsar. Si falla 3 veces, bloquea la cuenta.

Sistema crítico. Usaremos manejo de excepciones en cada intento. Una bandera (flag) controlará si hubo éxito o se debe aplicar el protocolo de bloqueo.

  • Ciclo While basado en intentos.
  • try para leer el PIN. Si es correcto, break. Si no, resta intento.
  • El except también debe restar un intento.
  • Integración profunda de While, Try/Except, If/Else y Flags.
def main():
    secret_pin = 9999
    attempts_left = 3
    is_locked = True

    # Security protocol loop
    while attempts_left > 0:
        try:
            user_pin = int(input(f"Ingrese su PIN ({attempts_left} intentos restantes): "))
            
            if user_pin == secret_pin:
                print("Autenticación exitosa. Acceso concedido.")
                is_locked = False
                break
            else:
                print("PIN incorrecto.")
                attempts_left -= 1
                
        except ValueError:
            print("Alerta de Seguridad: Formato no válido registrado.")
            attempts_left -= 1

    # Final security check
    if is_locked:
        print("SISTEMA BLOQUEADO. Contacte al administrador TIC.")

main()
Ejercicio 7: Verificador de Contraseñas Fuertes

Pide una clave. Para ser segura debe: Tener más de 8 caracteres, contener al menos un número y contener al menos una letra. Muestra si es Segura o Débil.

Análisis de cadenas avanzado. Evaluaremos condiciones booleanas separadas iterando por cada caracter del string entregado por el usuario.

  • Tres banderas booleanas: largo_ok, tiene_numero, tiene_letra.
  • Ciclo FOR por la contraseña. Usa métodos de string para detectar números y letras.
  • Al final, un IF evalúa si las 3 banderas son verdaderas simultáneamente usando AND.
  • char.isdigit(): Retorna True si es número.
  • char.isalpha(): Retorna True si es letra.
def main():
    password = input("Ingrese una nueva contraseña: ")
    
    # Initialize validation flags
    length_ok = len(password) > 8
    has_number = False
    has_letter = False
    
    # Scan characters for security requirements
    for char in password:
        if char.isdigit():
            has_number = True
        elif char.isalpha():
            has_letter = True
            
    # Evaluate final security status using AND logic
    if length_ok and has_number and has_letter:
        print("Estado TIC: Contraseña Fuerte.")
    else:
        print("Estado TIC: Contraseña Débil. No cumple con las políticas de seguridad.")

main()
Ejercicio 8: Menú Interactivo (CLI Tool)

Crea un menú que se repita siempre hasta que el usuario digite '3'. Opción 1: Saludar. Opción 2: Mostrar versión. Opción 3: Salir. Valida entradas erróneas.

La base de cualquier software de consola o sistema en terminal. Un bucle infinito que controla el estado de la aplicación mediante ruteo con condicionales.

  • Imprime el menú dentro de un while True.
  • Pide la opción. Usa If/Elif para cada caso.
  • En el caso 3, usa break para apagar el sistema. Usa else para capturar opciones inexistentes (ej: 9).
  • No necesitas try/except si tratas la entrada como texto (string). Es más seguro y no requiere conversión para comparar menús.
def main():
    # Main application loop
    while True:
        print("\n--- MENÚ DEL SISTEMA ---")
        print("1. Ejecutar Saludo")
        print("2. Información del Sistema")
        print("3. Salir")
        
        # Read as string to prevent casting errors
        choice = input("Seleccione una opción: ").strip()

            print(">> Hola, bienvenido al entorno de desarrollo.")
        elif choice == "2":
            print(">> Versión 1.0. Construcción Segura.")
        elif choice == "3":
            print(">> Apagando sistema...")
            break # Exit the application
        else:
            print(">> Error: Opción no válida. Intente nuevamente.")

Ejercicio 9: Auditoría de Notas de Alumnos

Pide al usuario cuántas notas desea auditar. Valida esa cantidad. Luego, usa un For para pedir cada nota validando que esté entre 1.0 y 7.0. Calcula el promedio.

Integración de doble validación: Primero cantidad (Entero positivo), luego datos iterativos (Flotantes en rango cerrado). Máxima robustez.

  • Primero aseguramos N. Luego iteramos N veces.
  • En cada iteración, otro While asegura que la nota sea float y esté en rango matemático.
  • Acumulador va sumando. Al final divide por N.
  • Anidación de validaciones. Arquitectura de defensa en profundidad.
def main():
    # Enforce valid number of iterations
    while True:
        try:
            n_grades = int(input("¿Cuántas notas ingresará al sistema?: "))
            if n_grades > 0:
                break
        except ValueError:
            pass
        print("Debe ingresar un número entero mayor a cero.")

    total_sum = 0.0
    
    # Loop exactly n_grades times
    for i in range(n_grades):
        # Enforce valid grade constraints
        while True:
            try:
                grade = float(input(f"Ingrese nota {i+1} (Rango 1.0 - 7.0): "))
                if 1.0 <= grade <= 7.0:
                    total_sum += grade
                    break
                else:
                    print("Error: La nota está fuera del rango permitido.")
            except ValueError:
                print("Error: Formato decimal incorrecto. Use un punto.")

    average = total_sum / n_grades
    print(f"\nAuditoría Finalizada. Promedio del alumno: {round(average, 1)}")

main()
Ejercicio 10 (Desafío Final): La Máquina de Café Robusta

Café = $1200. El usuario inserta monedas. Valida que el monto ingresado sea un entero. Usa un bucle para sumar las monedas hasta que alcance o supere los $1200. Imprime el vuelto exacto a entregar.

Simulación de hardware. El programa no avanza (no entrega producto) hasta que el requerimiento de negocio (precio) se cumple. Tolerante a caídas por mala inserción.

  • Acumulador saldo = 0.
  • while saldo < 1200: pide dinero, valídalo con try/except y súmalo.
  • Una vez que el ciclo termina (se cumplió la cuota), calcula el vuelto restando 1200 al saldo total.
  • Integración final de Variables, Acumuladores, While de Condición Comercial y Try/Except.
def main():
    coffee_price = 1200
    current_balance = 0
    
    print(f"--- MÁQUINA DE CAFÉ --- Precio: ${coffee_price}")

    # Transaction loop continues until fully funded
    while current_balance < coffee_price:
        missing = coffee_price - current_balance
        print(f"Saldo actual: ${current_balance}. Faltan: ${missing}")
        
        try:
            # Process incoming coin/bill
            inserted = int(input("Inserte dinero: $"))
            if inserted > 0:
                current_balance += inserted
            else:
                print("Hardware error: Moneda no reconocida.")
        except ValueError:
            print("Hardware error: Sensor atascado, ingrese un número.")

    # Transaction complete logic
    change = current_balance - coffee_price
    print("\n[+] Preparando su café...")
    
    if change > 0:
        print(f"[+] Dispensando vuelto: ${change}")
        
    print("[+] Operación Finalizada con Éxito. Gracias por su compra.")

main()
